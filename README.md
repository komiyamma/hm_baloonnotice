このプログラムは、Windows Vista ～ Windows 8.1 までで利用できるトースト通知のためのプログラムです。  
秀丸のみならず、汎用目的でコンソールからでも利用可能です。

Windows 10 以降は、<a href="https://秀丸マクロ.net/?page=nobu_tool_hm_baloontoast">HmBaloonToast</a> を利用してください。
